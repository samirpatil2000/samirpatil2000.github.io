Features

Fast workflows,
beautifully simple.

🏎️
Variable Speed

Smoothly adjust playback speed from 0.1x to 10.0x with precision controls.

📤
Instant Export

Baked-in speed changes exported directly to high-quality MP4 in seconds.

🎯
Drag & Drop

No complex menus. Just drop a file and start adjusting. Built for speed.

⚡
Hardware Accelerated

Utilizes macOS AVFoundation for super-fast processing and smooth previews.

✨
Native Performance

A primary macOS app designed to feel right at home on your desktop.

🔒
Fully Private

On-device processing. Your videos never leave your Mac. No cloud involved.
